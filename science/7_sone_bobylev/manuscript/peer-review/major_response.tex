\begin{quoting}
Необходимо добавить сравнение с данными~\cite{Sone1996}.
\end{quoting}
В работе~\cite{Sone1996} кроме прочего в качестве численного примера изучается рассматриваемая задача
с трёх сторон:
\begin{itemize}
    \item решение уравнений SNIT с коэффициентами для модели твёрдых сфер,
    \item решение уравнений SNIF с коэффициентами для модели БКВ,
    \item решение модельного уравнения БКВ.
\end{itemize}
Сравнение результатов уравнений Больцмана и БКВ для \(k=0\) также проведено в~\cite{Sone1996}.
Дополнительное их сравнение для конечных \(k\) видится автору излишним.
Касательно решения уравнений SNIF для модели твёрдых сфер, то авторские результаты
полностью совпадают с представленными в~\cite{Sone1996}, о чём дополнительно я в явной форме указал
в новой версии.

\begin{quoting}
На второй стр. сравниваются метод Чепмена-Энскога (МЧЭ) и метод Гильберта (МГ).
МЧЭ "--- частный случай весьма продвинутого и эффективного метода нелинейной теории возмущений:
метода сокращения информации, обладающего гораздо более широкой областью применимости,
чем методы типа МГ.
\end{quoting}
Действительно, метод сокращения информации позволяет получать возмущённые решения во многих приложениях,
однако в работе особый интерес представляет именно асимптотическое поведение решения,
полученного с помощью уравнений SNIF. По этой причине разложение Гильберта было взято за основу анализа.

Я скорретировал абзац, посвящённый сравнению методов, следующим образом.
\begin{leftbar}
В работах~\cite{Kogan1970,Kogan1971,Kogan1976} уравнения SNIF были получены наиболее простым способом,
на основе разложения Чепмена"--~Энскога, а необходимые транспортные коэффициенты были
вычислены в первом приближении с помощью полиномов Сонина~\cite{Chapman1960}.
Строгий асимптотический анализ уравнения Больцмана для медленных неизотермических течений
на основе разложения Гильберта был впервые выполнен в~\cite{Sone1996},
а транспортные коэффициенты были пересчитаны прямыми численными методами с высокой точностью.
Несмотря на то что оба метода приводят к одинаковым системам уравнений ведущего порядка,
подход Гильберта позволяет изучить асимптотические свойства решения уравнений SNIF.
В частности, в~\cite{Sone1996} было показано, что в континуальном пределе (\(\Kn\to0\))
скоростное поле первого порядка конечным образом влияет на температурное распределение нулевого порядка.
Подобные \emph{призрак-эффекты} не проявляются в уравнениях Навье"--~Стокса с граничным условием без скольжения,
но впоследствии были обнаружены при асимптотическом анализе ряда других задач~\cite{Sone2007}.
\end{leftbar}


\begin{quoting}
Уравнения МНТГ получены в~\cite{Kogan1970,Kogan1971,Kogan1976} с относительной погрешностью \(\OO{k^2}\),
где \(k\) "--- число Кнудсена...
\end{quoting}
По правде говоря, мне не очень понятно, что означает относительная погрешность дифференциальных уравнений.
По всей видимости, имеется ввиду второй порядок разложения по методу Чэпмена"--~Энскога,
т.\,е. учёт членов порядка \(\OO{k^2}\).
Но дело в том, что ввиду особого метода жонглирования членами различного порядка по \(k\)
в разложении Чэпмена"--~Энскога, невозможно судить о порядке аппроксимации
функции распределения и макроскопических величин при решении систем уравнений,
получаемых при различных порядках возмущения. Проясняет ситуацию разложение Гильберта,
из которого видно, что уравнения SNIF позволяют определить поле температуры с погрешностью \(\OO{k}\),
поле скоростей с погрешностью \(\OO{k^2}\), давление с погрешностью \(\OO{k^3}\),
а значит восстановить функцию распределения с погрешностью \(\OO{k}\), т.е. не точнее,
чем локальное распределение Максвелла.


\begin{quoting}
Представлялось хорошо известным, что при наличии температурного поля конвекция газа может изменять
